Simple Modal Example

Modal opens when you

At it's simplest, a modal may be triggered on event, and the content may come from a DOM element on the page.

A <style> block is set in the <head> of this page to hide the #modal_content div. At the end of <body> <script> blocks include build/das.js and create the modal.

This example uses the stylesheet at css/das.css, but you can just as easily use your own or write the CSS into the page. When specifying the optional stylesheet: argument, Das will append a style <link> element as the last element in the <head>.
